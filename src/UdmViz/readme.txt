------------
$Udm/src/UdmViz 
------------

This folder contains the UdmViz utility project.
UdmViz generates a textual representation from a Udm backend file to the standard output 
that serves as an input of the dot.exe from ATT GraphViz graph displaying utility 
(it can be downloaded from http://www.research.att.com/sw/tools/graphviz/download.html).
