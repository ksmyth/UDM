------------
$Udm/src/UdmBase
------------

This folder contains the UdmBase core library, including:
	- core UDM classes   (UdmBase.cpp)
	- Udm TOMI interface (UdmTomi.cpp)
	- UML helper classes (UmlExt.cpp)
	- UdmProject support (UdmProject.cpp)
