------------
$Udm/src/UdmUtil 
------------

This folder contains the UdmUtil library project.
Currently UdmUtil provides:
- function to copy an object hierarchy
- function to copy links from an object hierarchy to another
- function to extract name-like attributes from a generic Udm Object
