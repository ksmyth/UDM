------------
$Udm/src/UdmToXmi
------------

This folder contains the UdmToXmi binary utility project.

UdmToXmi does the conversion:
	- from:	MOF's XML representation of UML class diagrams (XMI)
	- to: UDM's XML representation of UML class diagrams
