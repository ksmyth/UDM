
New "@DSPACKAGE@" data type for Dashboard
===========================================

Copying the @DSPACKAGELOWER@_type-@VERSION@.nbm into the 
<your_dashboard_home>/modules directory will register 
the new data type "@DSPACKAGE@ XML Text File (.xml)" 
in Dashboard. The data type will be loaded if the 
@DSPACKAGELOWER@_type-@VERSION@.jar is also copied into the 
<your_dashboard_home>/modules directory.
