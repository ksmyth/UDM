-------------------
The description of files under $udm/include
-------------------

cint_string.h	-	the cint_string class definition
ErrHand.h	-	Udm::udm_exception class definition
UdmBase.h	-	core UDM header file (Object, ObjectImpl, DataNetwork classes
UdmCint.h	-	the classes UdmPseudoObject and UdmPseudoObjectS
UdmDom.h	-	DOM Backend definitions (DomDataNetwork, DomObject)
UdmGme.h	-	GME Backend definitions (GmeDataNetwork, GmeObject)
UdmInt.h	-	Udm interface to Cint C++ interpreter
UdmOcl.h	-	Object Constraint Language Implementation For UDM 
UdmProject.h	-	the UdmProject:: namespace (generated file, from $etc\UdmProject.xml)
UdmStatic.h	-	MEM backend definitions (StaticDataNetwork, StaticObject)
UdmUtil.h	-	Widely used UDM utility functions (copying an object hierarchy)
UdmXmi.h	-	functions for transforming from/to UDM/XMI Uml representations
Uml.h		-	The Uml:: API (generated file, from $etc\Uml.xml)
UmlExt.h	-	A set of UML helper functions fr obtaining ascendants, descendant, containerclasses, contained classes, etc.
